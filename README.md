
# Vue 3 + Vite

Esta plantilla ayuda a iniciarnos en el desarrollo con Vue 3 en Vite. La plantilla utiliza SFCs de Vue 3 `<script setup>`, consulta la [documentación de script setup](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) para aprender más.

Aprende más sobre el soporte de IDE para Vue en la [Guía de Escalado de Documentación de Vue](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).

### Resumen del Proyecto

La aplicación cumple con todos los requisitos especificados en el enunciado de la práctica:

*   **Tecnología**: Desarrollado con el ecosistema **Vue 3** (Vite, Pinia, Vue Router).
*   **API**: Integración completa con **PokeAPI** para recuperar datos dinámicos.
*   **Funcionalidades Principales**:
    *   Vista de **Lista de Pokemons** con filtrado en tiempo real.
    *   Vista de **Detalle** con estadísticas, tipos e imágenes.
    *   Vista de **Combate** (Opcional) implementada con lógica de juego de cartas y animaciones.
    *   Selección de **Tema** (Claro/Oscuro/Sistema) con persistencia en el navegador.
